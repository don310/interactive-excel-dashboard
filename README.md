# Store Annual Report 2026 - Excel Interactive Dashboard

## Project Overview
This project is an interactive data analysis dashboard built using Microsoft Excel.  
The dashboard provides insights into store performance, sales distribution, order status, and customer demographics for the year 2026.

## Dashboard Features
- Interactive filters using slicers
- Sales and order analysis
- Gender-based sales comparison
- Order status distribution
- Sales by top performing states
- Sales by online channels
- Age vs gender order analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization
- Excel Slicers

## Key Insights
- Women contributed **64% of total sales**
- Most orders were **Delivered (92%)**
- **Amazon and Myntra** were top performing sales channels
- **Maharashtra and Karnataka** generated the highest sales

## How to Use
1. Download the Excel file from this repository.
2. Open it using Microsoft Excel.
3. Use the slicers (Month, Channel, Category) to filter data.
4. Analyze trends using the charts and graphs.
